Part D - README Documentation

1. Project Description
This script is called analysis.py. It takes a number as input
from the command line, prints the number, and prints its squared value.

2. How to Run
First Create the scripts folder:
mkdir ~/hpc_qualifier/scripts

Now create the analysis.py file (paste or type the codee in the termina) in the scripts folder:
nano ~/hpc_qualifier/scripts/analysis.py

Navigate to the scripts folder first:
cd ~/hpc_qualifier/scripts

Then run the script with a number:
python3 analysis.py 5
python3 analysis.py 12

3. Results
Run 1:
python3 analysis.py 5
Input: 5
Squared: 25

Run 2:
python3 analysis.py 12
Input: 12
Squared: 144

4. Reflection
I learned that reproducibility means running the same script with
the same input should always give the same output. Documenting the
exact commands and results in a README makes it easy for anyone
to repeat and verify your work.
